上传要求：

  • 1. 请使用jpg\jpeg\png等格式、单张大小不超过4M的正方形图片。
  • 2. 上传图片最大尺寸将被保留为1280像素。
  • 4. 通过更改排序数字修改商品图片的排列显示顺序。
  • 5. 图片质量要清晰，不能虚化，要保证亮度充足。
  • 6. 操作完成后请点下一步，否则无法在网站生效。

建议:

  • 1. 主图为白色背景正面图。
  • 2. 排序依次为正面图->背面图->侧面图->细节图。
